To run App:
1. open Terminal and run socket server: Tutorial-Kivy-chat-app>python socket_server.py
	Info: the socket server listens to IP = "127.0.0.1" PORT = 50490
2. open Terminal and run the client App: Tutorial-Kivy-chat-app>python Tutorial-kivy.py
	Connection Screen: Enter for IP 127.0.0.1 for PORT 50490 and a username
3. open Terminal and run another client App: Tutorial-Kivy-chat-app>python Tutorial-kivy.py
	Connection Screen: Enter for IP 127.0.0.1 for PORT 50490 and a username
4. start chatting with the Client Apps.
